## 📝 Notes for Reference

### File Naming Convention

- `.jsx` files = React components
- `.css` files = Styles
- `.md` files = Documentation (this!)

### Where to Make Changes

- **Content (text):** Edit `.jsx` files
- **Styling (colors/layout):** Edit `.css` files
- **Images:** Replace in `public/` folder
- **Routes/Navigation:** Edit `AnimatedRoutes.jsx` and `Navbar.jsx`

### What NOT to Change

- Don't modify `package.json` dependencies (unless you know what you're doing)
- Don't change `vite.config.js` unless needed for deployment
- Don't delete existing components (customize instead)
